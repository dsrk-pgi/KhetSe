Important Tips for You:
File Names: I noticed some of your files have spaces (e.g., ladies finger.jpg) and mixed capitalization (e.g., POTATOS.png). The prompt above handles this, but if you rename them in your folder later, the app will show a broken image.

Case Sensitivity: Coding is sensitive to capital letters. Make sure POTATOS.png in the folder is exactly POTATOS.png (not potatos.png) in the code.

The "Public" Folder: In React/Next.js, anything in the public folder is accessed using a simple / path. So /public/images/beans.jpeg becomes just /images/beans.jpeg in the code.
